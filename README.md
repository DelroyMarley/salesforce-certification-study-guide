# Salesforce Certification Study Guide

An interactive quiz application for studying Salesforce certification exams.

## Features

- **Interactive Quiz Interface** - Practice with multiple-choice questions
- **Instant Feedback** - See explanations for correct answers
- **Multiple Exam Tracks** - Study for different Salesforce certifications

## Available Exams

| Exam | File | Questions |
|------|------|-----------|
| Sharing & Visibility Designer | `sharing-visibility.json` | 50+ |
| Platform Developer I | `platform-dev-1.json` | 50+ |
| Data Architecture Designer | `data-architecture.json` | 50+ |

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Select an exam and start practicing!

Or run `launch-quiz.bat` on Windows to start a local server.
